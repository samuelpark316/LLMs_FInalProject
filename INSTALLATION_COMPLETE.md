## 🐛 Troubleshooting

### Server Won't Start?
```bash
npx kill-port 3000
cd slack-mvp && npm run dev
```

### Changes Not Showing?
```bash
# Clear Next.js cache
rm -rf slack-mvp/.next
cd slack-mvp && npm run dev
```

### Module Errors?
```bash
cd slack-mvp
rm -rf node_modules package-lock.json
npm install
```
